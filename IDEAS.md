# Ideas / Backlog

Loose notes on improvements that aren't worth doing right now but worth remembering.

## UI / UX

- **Campaign-level date range that propagates to all adsets.** Each adset card has its own date range filter (14d/30d/etc), but in practice they're almost always set to the same value — which means the user has to update N controls to change scope. A single campaign-level filter would reduce duplicate UI controls and clicks. The per-adset filter could remain as an override, or be removed entirely if it's never used.
